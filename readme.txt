Bulgaria Map plugin for creating an interactive map of Bulgaria.

== Description ==
* Powerful plugin for creating an interactive map of Bulgaria.
* Map of Bulgaria with 28 clickable provinces.
* Include with shortcode and display the map on your WordPress website in any widget, post and page.
* Suitable for news sites, real estate sites, and all types of local businesses.

== Installation ==

= Install via Plugins > Install New =
1. Search for "BG Map"
2. Click the "Install Now" link
3. Click "Activate Plugin"

= Via ZIP / FTP =
1. Unzip the ZIP file and drop the folder straight into your wp-content/plugins directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
